Description of the The fiber optic handpiece used with the CoolTouch Varia laser
CoolTouch Corporation system has had a fixed spot size. The user selects a 4mm,
VariZoom handpiece: 6mm, or 8mm spot size, depending on the application. The
VariZoom handpiece combines the various spot size handpieces
into one, allowing the user to dial the desired spot size instead of
changing handpieces. The VariZoom handpiece has the same
coolant capability and incorporates a sensor to alert the user that
the coolant is depleted.
